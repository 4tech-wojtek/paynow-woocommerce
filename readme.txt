The official Paynow Plugin for WooCommerce - adds payment via BLIK and quick bank transfers

== Description ==
A new payment method that allows you to quickly and conveniently accept payments from customers. Simple configuration, BLIK payment and quick bank transfers. Withdrawal fund immediately after the transaction and all this for just a 0.95% fee. The lowest commission on the market and the fastest withdrawal to your account. Do not wait! Try it today.

== Installation ==

= Minimum Requirements =

* PHP version 5.6 or greater (PHP 7.0 or greater is recommended)
* MySQL version 5.6 or greater
* WordPress 4.4+
* WooCommerce 3.0+

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of WooCommerce Payment Gateway, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “Paynow Payment Gateway” and click Search Plugins. Once you’ve found our WooCommerce Payment Gateway plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading our plugin and uploading it to your server. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
